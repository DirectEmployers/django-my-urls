BaseX¶

The BaseX module encodes integers with a arbitrary alphabet.

class myurls.basex.BaseX(number=0, encoded='', character_set='')¶

Encodes and Decodes integers using a supplied character set

Attributes:

  • number – An integer
  • encoded – Encoded string
  • character_set – String containing character set used to encode
